# Exploring AI and Data: A Journey of Learning and Discovery

Welcome to your journey into the fascinating world of Artificial Intelligence and Data! In this document, you'll find a starting point to use as a list of resources to help you get started, but you’ll find tonnes more to enable you to dive deep into these exciting fields. Whichever lens you’re looking to explore these topics through (**Create**, **Implement**, **Solidify**, or **Custom**), these resources will provide you with a great starting point and overall view on direction - but they only scratch the surface… more great resources and directions come out every day!

### Large Language Models (LLMs)

- [Claude](https://claude.ai/): An AI assistant powered by Anthropic's large language model.
- [ChatGPT](https://openai.com/): OpenAI's powerful language model for conversational AI.
- [Llama](https://llama.meta.com/llama3/): Meta's large language model for natural language understanding.
- [Mistral](https://mistral.ai/): An AI writing assistant powered by large language models.
- [Gemini](https://gemini.google.com/app): Google's AI platform for conversational experiences - [get started building](https://ai.google.dev/).

Take a look at the open-source offerings from these model vendors, whether that’s the complete model like Meta’s Llama 3, or helpful programs and bots like [OpenAI’s Security Bots](https://github.com/openai/openai-security-bots). Each vendor will also have helpful docs and guidelines that will be helpful to navigate their APIs if you want to explore, such as [OpenAI’s cookbook](https://cookbook.openai.com/). Bear in mind closed source models likely cost to access, whilst open source can be accessed for free (although explore for yourselves).
